Description détaillée

Classe qui permet de gérer l'ensemble des notes.

La classe stocke l'ensemble des notes, permet leur modification, suppression, ainsi que la création de nouvelles notes. Toute modification/suppression/création de note doit obligatoirement passer par le NotesManager.
