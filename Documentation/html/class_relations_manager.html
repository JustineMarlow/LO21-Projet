Description détaillée

Permet la gestion de l'ensemble des relations.

La classe stocke l'ensemble des relations, permet leur modification, suppression, ainsi que la création de nouvelles relations. Toute modification/suppression/création de relation doit obligatoirement passer par le RelationsManager.
